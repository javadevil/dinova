# ฐานข้อมูลคำแนะนำการใช้ปุ๋ย N-P-K (ฉบับสมบูรณ์ - ชุดที่ 1)

## 1. เกณฑ์พิจารณาระดับความอุดมสมบูรณ์ของธาตุอาหาร (Soil Nutrient Thresholds)
*ใช้แปลงค่าตัวเลขจากผลวิเคราะห์ดิน เป็นระดับ (ต่ำ/กลาง/สูง) เพื่อใช้หาค่าปุ๋ย*

### 1.1 กลุ่มพืชไร่ (ข้าว, ข้าวโพด, มันสำปะหลัง, อ้อย, ถั่ว, ผัก, ยางพารา)
| ธาตุอาหาร | ระดับ: ต่ำ (Low) | ระดับ: ปานกลาง (Med) | ระดับ: สูง (High) |
|---|---|---|---|
| อินทรียวัตถุ (OM) | ข้าว/ข้าวโพด: < 1.0 | ข้าว/ข้าวโพด: 1.0-2.0 | ข้าว/ข้าวโพด: > 2.0 |
| | มัน/อ้อย: < 0.6 | มัน: 0.6-1.0 | มัน: 1.0-2.0 |
| | | | มัน: > 2.0 |
| ฟอสฟอรัส (P) | ข้าว/ข้าวโพด: < 5 | ข้าว/ข้าวโพด: 5-10 | ข้าว/ข้าวโพด: > 10 |
| | มัน/อ้อย: < 5 | มัน: 5-30 | มัน: > 30 |
| | ถั่ว/ผัก: < 8 | ถั่ว: 8-12 | ถั่ว: > 12 |
| โพแทสเซียม (K) | ข้าว: < 60 | ข้าว: 60-80 | ข้าว: > 80 |
| | ข้าวโพด: < 60 | ข้าวโพด: 60-100 | ข้าวโพด: > 100 |
| | ยางพารา: < 40 | ยางพารา: > 40 | - |

### 1.2 กลุ่มไม้ผล (มะม่วง, ทุเรียน, มังคุด, ฯลฯ)
| ธาตุอาหาร | ระดับ: ต่ำ (Low) | ระดับ: ปานกลาง (Med) | ระดับ: สูง (High) |
|---|---|---|---|
| อินทรียวัตถุ (OM) | < 1.5 | 1.5 - 2.5 | > 2.5 |
| ฟอสฟอรัส (P) | < 15 | 15 - 45 | > 45 |
| โพแทสเซียม (K) | < 50 | 50 - 100 | > 100 |
